# Benthos
R scripts of an ecological study\n
Functioal diversity and species compostion of benthic invertebrates of mangroves of Iran. /n
A Project in [naderloolab.com ](https://naderloolab.com/), University of Tehran.
Incoludes multivariate analysis (PCA, nMDS, SIMPER), visualization, hypothesys testeing (PERMANOVA, ANOVA, post-hoc), and functial and species diversity indices.
