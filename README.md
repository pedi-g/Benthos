# Benthos
R scripts of an ecological study.

Functioal diversity and species compostion of benthic invertebrates of mangroves of Iran.

A Project from [naderloolab](https://naderloolab.com/), University of Tehran.

Incoludes multivariate analysis (PCA, nMDS, SIMPER), visualization, hypothesys testeing (PERMANOVA, ANOVA, post-hoc), and functial and species diversity indices.
